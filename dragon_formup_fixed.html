📖 Dragon Formup Documentation
🖌️ Drawing Tools
Brush — Paint with soft edges. Adjust Size, Hardness, Opacity, Flow in the left panel. Use Blend Modes for creative effects.
Pencil — Hard-edge drawing tool, pixel-perfect lines.
Eraser — Erase pixels. Works with brush size/opacity settings.
Clone Stamp — Duplicate pixels from one area to another.
Healing Brush — Remove blemishes and imperfections.
Dodge — Lighten specific areas of the image.
Burn — Darken specific areas of the image.
Smudge — Blend and smear colors together.
🎨 Color Tools
Paint Bucket — Flood fill an area with the foreground color.
Eyedropper — Click anywhere to pick that color as foreground.
Gradient — Draw linear gradient from FG to BG color.
Color Wheel — Click the wheel in Colors tab to select any hue. Use Hex input for precise colors.
📐 Shape & Selection
Rectangle Select — Draw a rectangular selection area.
Lasso — Free-form selection by drawing around an area.
Magic Wand — Select similar-colored regions automatically.
Shapes — Draw Rectangle, Ellipse, Line, Triangle, Polygon.
⚡ Adjustments Panel (Right Panel → Adjust)
Sliders for Exposure, Brightness, Contrast, Highlights, Shadows, Whites, Blacks, Saturation, Vibrance, Hue, Temperature, Tint, Sharpness, Noise Reduction, Clarity, Dehaze, Vignette, Grain.
Click ✓ Apply to commit changes. Click ↺ Reset to revert all sliders.
✨ 50 Filters (Right Panel → Filters)
Classic: Grayscale, Sepia, Invert, Vintage, Blueprint, Old Photo
Enhance: Vivid, Punch, Clarity+, HDR, Detail+, Pop Color
Instagram: 1977, Aden, Clarendon, Gingham, Inkwell, Lo-Fi, Toaster, Valencia, X-Pro II, Moon
Artistic: Oil Paint, Watercolor, Pencil Sketch, Charcoal, Comic, Pastel, Glitch, Neon Glow
Cinematic: Cinematic, Film Noir, Teal Orange, Golden Hour, Cool Blue, Warm Sunset, Matte, Faded
Technical: Sharpen, Gaussian Blur, Motion Blur, Radial Blur, Emboss, Edge Detect, Denoise, Unsharp Mask
Color Grading: Cross Process, Split Tone, Solarize, Duotone R/B, Heatmap, Chrome
Special FX: Pixelate, Mosaic, Halftone, Scanlines, VHS Effect, Infrared, Lomo, Dither
🗂️ Layer System (Right Panel → Layers)
Add, Duplicate, Delete layers. Set Opacity and Blend Mode per layer.
Use Merge Down to combine two layers, or Flatten to merge all into one.
💾 Saving & Export
File → Save as PNG / JPG / WebP. PNG preserves transparency. JPG is smallest file size. WebP is best for web use.
